Welcome to my Kotlin repository! This project serves as a comprehensive record of my journey learning **Kotlin**. Here, I practice core programming concepts, object-oriented principles, and functional programming features offered by the language.

---

## 📚 Topics Covered

I have structured my learning into several fundamental and advanced topics:

### ⚙️ Fundamentals & Basics
* **Variables, Constants, & Types:** Understanding `val` (read-only), `var` (mutable), and Kotlin's static typing system.
* **Strings:** Working with string manipulation, templates, and raw strings.
* **Conditionals:** Using `if-else` expressions and the powerful `when` statement.
* **Loops:** Implementing iteration using `for`, `while`, and `do-while` loops.

### 🧩 Data Structures & Collections
* **Arrays and Lists:** Managing ordered sequences of data (both mutable and immutable).
* **Sets:** Working with collections of unique elements.
* **Maps:** Using key-value pairs for efficient data lookups.
* **Enumerators (Enums):** Defining fixed sets of constants representing state or choices.

### 🛠️ Functions & Advanced Collections
* **Functions:** Writing clean, reusable blocks of code, including named arguments and default values.
* **Functional Operators:** Transforming and filtering data efficiently using:
    * `map`
    * `filter`
    * `reduce`

### 🛡️ Object-Oriented Programming & Safety
* **Classes:** Creating blueprints for objects with properties and methods.
* **Inheritance:** Extending functionality and promoting code reuse between classes.
* **Null Safety:** Utilizing Kotlin's type system to eliminate the dreaded `NullPointerException` using safe calls (`?.`) and the Elvis operator (`?:`).
